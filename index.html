What I've Learned

For this assignment, I explored new HTML and CSS features that weren't covered in class:

The <details> HTML Element

I implemented the <details> element, which creates a small widget that can be expanded to display additional information about myself.

The CSS filter Property

I used the filter CSS property to apply a drop-shadow effect to the profile picture. I noticed that it allowed for multiple visual effects like blur, brightness, & shadows without the need for external graphics editors. I also found that if I combined it with a transition, it resulted in a smooth, interactive effect when someone hovers over the image.

The CSS mix-blend-mode Property

For the interactive element, I implemented the mix-blend-mode CSS property. This implementation doesn't serve any practical purposes but I thought it was cool to look at. It determines how an element's content blends with that of its parent and background.

API Integration

I added the Metropolitan Museum of Art's public API to display artwork from their collection. This helped me refresh the practice of calling API's and updating the DOM with the response data.
